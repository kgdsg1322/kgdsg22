# 검증 체크리스트 전문 — 5항목 게이트

> 모든 판정은 이미지를 Read로 직접 열어 본 것에만 근거한다. 추측 판정 = 검증 실패보다 나쁜 오류다.

## V1. 설교문 정합 (날조 차단)

- [ ] 이미지의 모든 인물·사건·소품이 매핑표의 "설교문 대응 단락(원문 인용)"에 실재하는가
- [ ] 매핑표 인용문이 실제 설교문 원문과 일치하는가 (대조 확인)
- [ ] 설교문에 없는 등장인물·기적·소품이 추가되지 않았는가
- [ ] 장면 순서가 설교 흐름과 일치하는가

## V2. 신학·본문 정합 (정통 기준)

- [ ] 성경 본문이 명시하지 않은 사항을 단정 형상화하지 않았는가
      (전례: 선악과를 '사과'로 특정 — 성경 미명시. 중립 형상이어야 함)
- [ ] 성경 인물·사건의 묘사가 본문 내용과 모순되지 않는가
- [ ] 정통 기독교(개혁주의) 기준에서 오해를 부를 도상이 없는가
      (예: 하나님 성부의 직접 인격 형상화 — 빛·광선 등 간접 표현 권장)
- [ ] 설교문 자체의 신학 문제 발견 시: 수정하지 말고 ⚠️ 플래그로만 보고 (권한 밖)

## V3. 아동 안전 (1–7세)

- [ ] 공포 유발 요소 없음: 위협적 표정·어둠 속 고립·피·괴물형·날카로운 무기 강조
- [ ] 주인공이 밝고 안정적인가 (위험 장면도 보호·용기 중심으로 읽히는가)
- [ ] 과자극 없음: 형광색·과밀 구도·혼란스러운 패턴
- [ ] 슬픔·죽음 주제는 소망·위로의 톤으로 처리되었는가

## V4. 저작권 안전

- [ ] prompts/ 기록 전수 검사: 출판물·작가·브랜드·캐릭터 고유명사가 프롬프트에 없는가
- [ ] 기존 유명 작품·캐릭터로 식별될 만한 외형이 아닌가
- [ ] 이미지 안에 텍스트(글자·로고)가 없는가 (텍스트 제로 규칙)

## V5. 시리즈 일관성

- [ ] 동일 인물의 외형이 전 장면 동일한가: 비율·머리색·의상색·이목구비 (장면 간 직접 대조)
- [ ] 팔레트가 화풍 기준(난색 파스텔)에서 벗어난 장면이 없는가
- [ ] 선 굵기·질감·음영 처리가 통일되어 있는가
- [ ] 해상도 1280×720·JPG 규격 전수 확인 (`sips -g pixelWidth -g pixelHeight`)

## 판정 규칙

| 판정 | 조건 |
|------|------|
| PASS | 해당 항목의 모든 체크 통과 + 실측 근거 기록 |
| FAIL | 하나라도 위반 — 위반 내용·위치를 구체적으로 기술 |
| 미확인 | 실측 불가(파일 손상 등) — PASS로 처리 금지, 사유 기록 |

수정 지시서는 "재생성 프롬프트에 그대로 반영 가능한 수준"으로 구체적으로 쓴다.
(나쁜 예: "양이 이상함" / 좋은 예: "03번 양의 귀가 1·2번과 달리 뾰족함 — 둥근 타원 귀로, 심볼 #sheep 기준 준수")
